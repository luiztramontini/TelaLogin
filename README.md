# TelaLogin
Tela de login simples, apenas para teste e estudo. Criei apenas como base, um esboço do que está por vir. Coloquei apenas comandos cristalinos, como as mensagens "Login feito com sucesso!", para quando o acesso for aceito, e "Login incorreto" quando o acesso for negado, além de usar o asterisco "*" para ocultar a senha.
